# 共用记账网页

这是一个手机微信可打开的极简记账网页。当前功能包括：

- 记录支出金额、分类、支付人、日期、备注
- 账单列表和关键词搜索
- 月份筛选
- 支付人占比饼图
- 分类支出柱状图：每个分类同时显示上月和本月
- 首页显示本月支出、本月预算、预算剩余
- 本地保存
- 填入 Supabase 配置后多人云端同步

## 本地打开

直接打开 `index.html` 即可使用。手机微信访问时，需要把整个 `ledger-web` 文件夹部署到一个 HTTPS 静态站点。

## 云端同步

1. 创建一个 Supabase 项目。
2. 在 SQL Editor 里执行 `supabase-schema.sql`。
3. 打开网页的“设置”页，填写：
   - 我的名字
   - 共享账本 ID
   - Supabase 项目 URL
   - Supabase anon key
4. 把同样的网页地址、共享账本 ID、项目 URL 和 anon key 发给共用的人。

注意：这个版本追求简单共用，没有做用户登录。知道配置的人都能访问同一个账本。正式长期使用建议再加登录或服务端鉴权。
